Detailed Description

The downstream XGTC frame of XG-PON.

This class has fields corresponding to those in an XGTC downstream frame (the header and an array of packets/XGEM frames) as well as methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.
