Detailed Description

The class instantiates XgponOnuConnManager. It does impose several relationship between alloc-id, xgem-port, onu-id, ipaddress, etc.

  1. alloc-id values: (onu_id) + (0...15) * 1024. T-CONTs are organized by a vector whose length is 16.
  2. ds/us xgem-port values: onu_id + (0...63) * 1024. xgem-ports are organized by a vector whose length is 64.
  3. ipaddress: onuid + other bits + 6bits for identify each computer (xgem port) that belongs to this ONU. onu-id: 10bits; xgem-port: 16bits; alloc-id: 14bits Hence, for each ONU, the number of supported xgem-ports / t-conts must be less than 64 / 16 (should be enough for many scenarios).
