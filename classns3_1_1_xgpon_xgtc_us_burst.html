Detailed Description

The upstream XGTC burst of XG-PON.

This class has fields corresponding to those in an XGTC upstream burst (the header, the trailer, and an array of allocation that is an array of XGEM frames) as well as methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.
