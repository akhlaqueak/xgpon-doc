Detailed Description

The per T-CONT payload in the upstream XGTC burst.

This class has fields corresponding to those in the per T-CONT payload of the upstream XGTC burst. (the buffer occupancy report and an array of XGEM frames that belong to this T-CONT) as well as methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.
