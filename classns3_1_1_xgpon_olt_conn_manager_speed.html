Detailed Description

The class instantiates XgponOltConnManager. It imposees some constraint between address and onuId/XgemPort so that, the corresponding xgem-port can be calculated from address, and get the connection in O(1) time. It is optimized for uni-cast traffics. More specifically, DS XgemPort values: onuId + (0...63) * 1024; IP address: onuId + other bits + 6bits.
