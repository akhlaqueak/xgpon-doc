Detailed Description

The buffer occupancy report used by XG-PON for DBA.

This class has fields corresponding to those in a buffer occupancy report structure used by XG-PON's DBA scheme. This class also includes the methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.

Although there is no Alloc-ID in this class, OLT parses it according to the corresponding bandwidth allocation map, such as the timeslot for this Alloc-ID and the flag for asking queue occupancy report from the ONU.
