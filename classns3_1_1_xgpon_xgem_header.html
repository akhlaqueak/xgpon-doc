Detailed Description

The header for the XGEM frame used by XG-PON for encapsulating SDU from upper layers.

This class has fields corresponding to those in an XGEM header as well as methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.
