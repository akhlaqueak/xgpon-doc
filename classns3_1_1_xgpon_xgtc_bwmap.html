Detailed Description

The BW-MAP structure used by XG-PON.

This class mainly contains a list of XgponXgtcBwAllocation. used by XG-PON's DBA scheme. This class also includes the methods for serialization to and deserialization from a byte buffer. It follows ITU-T G.987.3 recommendation.
